Extends original wordpress importer to allow zipped xml files

== Description ==

Extends (and requires) original wordpress importer to allow zipped xml files

== Installation ==

1. Upload `wordpress-importer-zipped` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently asked questions ==

= Does it allows to upload larger files =

Yes and no.
It allows to upload xml zipped files, so the original file can be larger, but the total filesize allowed is the same.
